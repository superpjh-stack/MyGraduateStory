MyGraduate - AI graduate school learning tracker with storytelling focus.

Tech stack decided: Next.js 14+ (App Router), TypeScript, Supabase (PostgreSQL + Auth + Storage), Tailwind CSS + shadcn/ui, Zustand, TanStack Query, Recharts, Framer Motion, Vercel.

Project level: Dynamic (not Enterprise, not Starter).

PDCA Phase 1 (Plan) completed on 2026-03-13 with full product-requirements.md.
PDCA Phase 2 (Design) completed on 2026-03-13 with system-architecture.md, ui-ux-design.md, data-model.md.
Development plan created on 2026-03-13.

**Why:** Gerardo wants to track 52 weeks of classes and generate a graduation storybook at the end.
**How to apply:** All design decisions should prioritize simplicity (1-person dev), data safety (1 year of records is precious), and emotional design (warmth, growth metaphor).
